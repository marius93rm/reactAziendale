## Confini del laboratorio

Il Modulo 1 lavora su component composition, componenti presentazionali, container, custom hook, dipendenze, prop drilling, feature-based structure e naming. Il servizio locale asincrono serve a esercitare Effect, cleanup e dependency injection senza introdurre backend o credenziali.
